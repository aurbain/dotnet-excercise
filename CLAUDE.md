# Project Guidelines

## Development & Deployment

- Make more frequent calls to the LLM. It is running locally and is tight on RAM.
- Break tasks into sub-tasks where possible.

### Service Management
- Stop the application: `sudo systemctl stop HelloWorldWebApp.service`
- Start the application: `sudo systemctl start HelloWorldWebApp.service`
- Restart the application: `sudo systemctl restart HelloWorldWebApp.service`
- Check status: `sudo systemctl status HelloWorldWebApp.service`
